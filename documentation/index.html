Introduction

Material is a fully responsive admin web App built with Bootstrap & Angular. It uses Sass CSS which makes it easy to modify.

File Structure

Once downloaded, unzip the compressed folder and you'll see something like this:

How to Run the Demo?

Note: dist/ folder contains the built distribution (production) version, which is for demo purpose only

You need to put the files on a server to run it. Here we will talk about how to build a server locally.

You can build a locally server using WAMP or MAMP (if you uses PHP).

Also, it's pretty easy to build one with Python or Ruby (if you prefer Python or Ruby).

Here, we'll build a local server using NodeJS

Preparation:

You'll need to install Node.js >=v6.9.0 (NPM comes along with it) and node-static (a NPM package)


Steps:

It's easy if you are familiar with Node. Download Node.js here and install it, then install node-static package with one command line

npm install -g node-static

The usage of node-static is simple. Just enter the "dist" directory with command line

cd dist/

and type:

static

It'll return something like this: 

serving "." at http://127.0.0.1:8080

Then you can view the template on http://127.0.0.1:8080 with your browser

Note: Because the dist/ folder built with webpack uses absolute path, you need to run it from root path, if not, please compile a new dist/ with a new publicPath value and run `npm run build`.

How to Use It?

In order to do development with it, first you'll need have the following tools installed:

Those tools are for development purpose only, you don't need them in production environment, meaning you can use any back-end languages you like, NodeJS is just one of the options, not required

  • Node.js >=v6.9.0 (NPM comes along with it), just download and install it, that's it.
  • Ruby (if you are using a Mac, you already have it), Sass for Scss CSS.

    • Install Angular CLI globally:

    npm install -g @angular/cli

    A few steps to get started:

    1. Enter "material" folder with your command line tool.

    cd material

    2. Install all the NPM packages with command: 

    npm install

    3. Then type ng serve and you are ready to go :)

    ng serve --open

    4. It will open the admin web app automatically with your default browser, if not, go to http://localhost:4200 in your browser (Note: For the first time, it may takes a bit longer to load, about 20 seconds)

    Note: All the source code for development are inside ./material/src/ folder.

    Check out CLI Quickstart first to know the basics of Angular CLI

    Check out Angular CLI documentation to know more about Angular CLI


    Webpack

    If you prefer to use Webpack instead of Angular CLI, simply run command ng eject, it ejects your app and output the proper webpack configuration and scripts.

    ng eject

    Check out Webpack Getting Started guide or Webpack: an introduction for Angular to get up and running.
