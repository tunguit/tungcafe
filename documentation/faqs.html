Layouts & Color Schemes

You can simply update a few variables to change the default settings.

Open material/src/app/config.ts file, you'll find following code. You can change the value to update the default settings.

let AppConfig = {
    brand: 'Material',
    user: 'Lisa',
    year: year,
    layoutBoxed: false,                             // true, false
    navCollapsed: false,                            // true, false
    navBehind: false,                               // true, false
    fixedHeader: true,                              // true, false
    sidebarWidth: 'middle',                         // small, middle, large      
    theme: 'light',                                 // light, gray, dark
    colorOption: '34',                              // 11,12,13,14,15,16; 21,22,23,24,25,26; 31,32,33,34,35,36
};

For example you want to set "boxed" layout as default, then you can update code like so:

let AppConfig = {
    brand: 'Material',
    user: 'Lisa',
    year: year,
    layoutBoxed: true,                              // true, false
    navCollapsed: false,                            // true, false
    navBehind: false,                               // true, false
    fixedHeader: true,                              // true, false
    sidebarWidth: 'middle',                         // small, middle, large      
    theme: 'light',                                 // light, gray, dark
    colorOption: '34',                              // 11,12,13,14,15,16; 21,22,23,24,25,26; 31,32,33,34,35,36
};

Done and enjoy :)

How to remove a module

Here we take charts module as an example

Steps:

1. Open file material/src/app/layout/layout-routing.module.ts and remove following line to stop lazy load the ChartsModule

{ path: 'chart', loadChildren: '../charts/charts.module#ChartsModule' },

2. Remove material/src/app/charts folder to remove all the files


3. Open filematerial/src/app/layout/sidenav/sidenav-menu/sidenav-menu.component.html and remove charts related menu items

<li>
    <a md-button href="#/app/chart"><i class="nav-icon material-icons">equalizer</i><span class="nav-text">Charts</span></a>
    <ul>
        <li><a md-button class="md-button prepend-icon" [routerLink]="['/app/chart/line']">Line & Area</a></li>
        <li><a md-button class="md-button prepend-icon" [routerLink]="['/app/chart/bar']">Bar</a></li>

</li>

Done :)
